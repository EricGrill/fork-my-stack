# MCP Servers I Publish

Reference list of MCP servers from my ecosystem, with practical usage notes.

GitHub: <https://github.com/EricGrill>

---

## 1) mcp-proxmox-admin

**What it does:** Manage Proxmox infrastructure tasks through MCP tools.

**Install command:**
```bash
npm install -g mcp-proxmox-admin
```

**Use cases:**
- VM lifecycle operations
- Node/resource checks
- Basic homelab orchestration
