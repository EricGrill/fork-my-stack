# HEARTBEAT.md (Template)

When a heartbeat poll arrives, run this checklist in order.
If nothing needs attention, reply: `HEARTBEAT_OK`.

## Checklist

1. **Email**
   - Check unread priority messages
   - Flag anything urgent

2. **Calendar**
   - Check next 24-48 hours
   - Alert on upcoming events (especially <2h)

3. **Weather**
   - Check current + next few hours
   - Mention severe weather or travel-impacting conditions

4. **Project status**
   - Review active tasks
   - Surface blockers and next actions

5. **Custom checks**
   - Add your own checks here
   - Example: server health, mentions, deployment status

## Quiet Hours

- Avoid non-urgent pings during: YOUR_QUIET_HOURS
- Urgent alerts can bypass quiet hours

## Output format

- If action needed: concise summary + recommended next step
- If no action needed: `HEARTBEAT_OK`
