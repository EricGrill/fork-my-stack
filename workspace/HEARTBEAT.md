# HEARTBEAT.md (Template)

When a heartbeat poll arrives, run this checklist in order.
If nothing needs attention, reply with `HEARTBEAT_OK`.

## Checklist

1. Email
   - Check unread priority messages
   - Flag urgent items

2. Calendar
   - Check next 24 to 48 hours
   - Alert on near-term events and conflicts

3. Notifications
   - Check mentions or direct messages
   - Surface items needing response

4. Project status
   - Review active tasks
   - Highlight blockers and next actions

5. Optional custom checks
   - System health
   - Deploy status
   - Other user-defined checks

## Quiet Hours

- Avoid non-urgent pings during: YOUR_QUIET_HOURS
- Urgent alerts can bypass quiet hours

## Output Format

- If action is needed: short summary plus recommended next step
- If no action is needed: `HEARTBEAT_OK`
